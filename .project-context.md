# Project Context: pi-bg-process-windows

**Status:** v4.0 Modular + PTY — Complete & Tested
**Last Updated:** 2026-04-20

---

## Current State

v4.0 of the extension is fully restructured from a monolithic `index.ts` into 7 modules with PTY support via node-pty (ConPTY on Windows). The key architectural change is replacing the old spawn-only approach with a dual execution path: `executeWithSpawn` for fast sync commands and `executeWithPty` for interactive background commands.

The critical `deliverAs: "followUp"` fix ensures `[BG_DONE]` notifications arrive automatically — previously they used `"nextTurn"` which required the user to send a prompt before the agent would see them. All features have been tested interactively: auto-backgrounding, peek with scroll, input to interactive prompts, and automatic completion notifications.

## What This Project Is (v4.0)

A PI Coding Agent extension that shadows the built-in bash tool on Windows with:
- Auto-backgrounding for long-running commands (60s sync timeout)
- PTY mode (`sync: false`) for interactive processes — agent can peek and send input
- Automatic `[BG_DONE]` notifications via `deliverAs: "followUp"`
- `[BG_SILENT]` after 30s with no output — shows recent output instead of heuristic guessing
- Scrollable TUI log viewer (`/win_tasks`)
- Footer indicator showing running/finished process counts
